วิธีการติดตั้งแอพฟาร์มรักษ์

แนะนำวิธีการติดตั้งสำหรับแอพฟาร์มรักษ์สำหรับ Android ท่านสามารถสไลด์เพื่อดูหน้าถัดไปได้

1) ดาวน์โหลดแอพ

ท่านสามารถดาวน์โหลดแอพฟาร์มรักษ์ได้โดยกดที่ปุ่มด้านล่าง

กดดาวน์โหลด

2) ไปที่การแจ้งเตือน

โดยท่านสามารถสไลด์หน้าจอดด้านบนของท่านลงมา ท่านจะพบกับไฟล์ farmrak.apk จากนั้นให้ท่านกดไปที่ไฟล์ farmrak.apk

3) จะมีปุ่มให้ท่านกด Install

โดยท่านสามารถกด Install เพื่อทำการติดตั้งแอพฟาร์มรักษ์ลงที่เครื่องของท่าน

4) จะมีการขออนุญาตในการติดตั้งแอพ

ให้ท่านกดที่ปุ่มยืนยันเพื่อเปิด allow permission หากมีการปิดอยู่

5) หลังจากกดอนุญาต permission แล้ว

แอพฟาร์มรักษ์กำลัง Install หรือติดตั้งลงเครื่องโทรศัพท์ของท่าน

6) หากมีให้กดสแกนแอพ

ให้ท่านกดที่ปุ่ม Scan app เพื่อทำการตรวจสอบการติดตั้งของแอพฟาร์มรักษ์เพื่อเป็นการยืนยันความปลอดภัย

7) หากสแกนเสร็จเรียบร้อยแล้ว

ให้ท่านกดที่ปุ่ม install อีกครั้ง เพื่อทำการดาวน์โหลดต่อ

8) หากดาวน์โหลดสำเร็จแล้ว

จะปรากฏดังภาพให้ท่านกด Open เพื่อทำการเข้าใช้แอพฟาร์มรักษ์ของเราได้เลย
